# 2MateBot

## About
> **2Mate** is a bot specially created for
> Dusework's Discord server, but anyway
> it can be used within other Discord
> servers.

## Requirements
> This project requires Python 3.6 and above, first
```ini
Python==3.6
```

> Some of these helpful libraries for Python
```ini
enums==0.0.2
discord.py==1.4.1
```
